Geometric hover effects with math

DOM Geometry

All the hover effects begin with getBoundingClientRect(), which is used to calculate the center point of each element and how far the cursor is from that point.

Pythagorean Theorem

The distance calculation uses the Pythagorean theorem to determine how far your mouse is from the element's center.

Two-Element Arctangent

The code uses Math.atan2() to calculate the angle between the mouse and the element center.

Trigonometry in action

Given that angle, the effect then applies rotations using Math.sin() and Math.cos().

3D transformations

CSS perspective and the rotateX and rotateY properties create the illusion of depth as you move your mouse around.

Responsive!

If the viewport size changes, the code makes a new call to getBoundingClientRect() to account for each element's new size.
